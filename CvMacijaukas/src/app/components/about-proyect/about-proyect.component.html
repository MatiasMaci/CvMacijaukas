PortFolio de Matias Macijaukas


Se trata de una simple SPA creada y diseñada para incluir información sobre mi currículum.
(Tengo un Portfolio Web utilizando POO, JavaScript para el front, Java para el back, sistema de Login y base de datos, pero con las reiteradas caídas de Render, resultaba complicado y tedioso esperar su carga al entrar en la página. Asi que opté por realizar un simple portfolio utilizando únicamente HTML, CSS y TypeScript).
